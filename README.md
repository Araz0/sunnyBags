# sunnyBags

## about

This is a side project aimed to create a page to showcase diy hand bags.
It should show a gallery of bags, and when clicked on one bag, you can see more information about it.

## specs

every bag has meta information, like:

- id
- name
- category
- describtion
- front and back photos
- fabric
- size
- price
- discount in %
- bg color? (if thumbnails are png, we can custimize backgrounds)
